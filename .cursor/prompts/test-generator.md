## 手順（フォルダ管理付き）
1. **フォルダの確認/作成**:
   - Google Drive上に「プロジェクト名（例：famihon-app）」というフォルダがあるか確認する。
   - なければ新規作成し、その **Folder ID** を取得する。
2. **ファイルの作成**:
   - 取得した Folder ID 内に、新しいスプレッドシート（テスト仕様書）を作成する。
3. **仕様書の書き込み**:
   - 以降、そのスプレッドシートに対してテストケースを書き込む。

4. **権限・操作ができない場合**:
   - 権限やファイル・フォルダの操作ができない場合は、必要な操作内容を具体的に伝え、ユーザーへ依頼する。
